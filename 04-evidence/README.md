# 04 — Evidence

Verified provenance. This is the anti-drift core of the repo: every canonical number
about the live system originates here and is cited elsewhere, never retyped.

- `INFRASTRUCTURE_SNAPSHOT.md` — the consolidated verified snapshot (point-in-time)
- `scripts/` — the read-only audit scripts that produce the numbers
- `snapshots/` — dated raw artifact folders (or pointers to them on the host)

**Regenerate, don't edit.** See ../CONTRIBUTING.md.

governance program sponsored by Harmony for Hope, Inc. (501(c)(3)) — and **Ms. Allis**,
the self-hosted community AI that supports it.

> **Ms. Allis** — *ALLIS: Artificial Learning & Location Intelligence System.*
> "Ms. Allis" / "ALLIS" is the name used everywhere in this repository. The live
> infrastructure uses the internal service prefix `jarvis-` (e.g. `jarvis-chroma`);
> that prefix is the engine-room namespace, not a second name.

## How this repo relates to the work repo

This is the **audited storefront**: small, verified, single-named, point-in-time.
The full lab notebook — the 42-chapter thesis, methodology, evaluations, and the
complete development record — lives in the work repo and is the provenance source.
See [`90-provenance.md`](90-provenance.md).

## The two rules that keep this repo honest

1. **Numbers are regenerated, never hand-typed.** Every canonical figure lives in
   [`04-evidence/`](04-evidence/) and is produced by a dated audit script. Other
   documents *cite* it. There is structurally one container count, one vector count.
2. **Only stable, verified, forward-facing material lives here.** Anything in
   progress stays in the work repo and is cross-linked.

See [`CONTRIBUTING.md`](CONTRIBUTING.md) for both in full.
